 Dashboard RH  
 
Ce projet propose un dashboard RH clair et moderne, conçu pour offrir une vue d’ensemble rapide des informations relatives aux employés.

Fonctionnalités

Visualisation du nombre d’employés par département

Calcul du salaire moyen des employés

Suivi du nombre d’employés dans le département IT

Historique des nouveaux recrutements par année

Graphiques interactifs réalisés avec Chart.js

Technologies utilisées

PHP pour récupérer et gérer les données

MySQL pour la base de données

HTML, CSS et Bootstrap 5 pour le design et la mise en page

Chart.js pour les représentations graphiques






This project provides a clean and modern HR dashboard, designed to give a quick overview of employee data.

Features

View the number of employees per department

Calculate the average salary of employees

Track the number of IT department employees

Monitor new hires by year

Interactive charts powered by Chart.js

Technologies Used

PHP for data retrieval and management

MySQL for the database

HTML, CSS, and Bootstrap 5 for design and layout

Chart.js for data visualization
